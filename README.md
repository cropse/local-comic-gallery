# local-comic-gallery
comic folder as static in root folder
```
static/
|--comic Name
|  |--01.jpeg
|  |--02.jpeg
|  |--03.jpeg
```
---
### Runing service 
***docker-compose required*** \
in .devcontainer folder \
`docker-compose up -d` \
open \
`localhost:3000`
